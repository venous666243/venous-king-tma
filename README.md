## Русский

**Venous King** — это высококонверсионное премиальное веб-приложение (Mini App) для Telegram, разработанное для ресторанов, стритфуда и служб доставки еды премиум-сегмента. Приложение открывается мгновенно внутри мессенджера в 1 клик.

### 🔥 Ключевые фичи:

* **Люксовый дизайн:** Интеграция UI-палитры Royal Deep Burgundy (глубокий бордовый фон, графические карточки, золотые акценты).

* **Премиальный UX:** Ультра-плавные микро-анимации, живой отклик кнопок `active:scale-95`) и динамический подсчет корзины.

* **Интеграция с Telegram:** Динамическое приветствие пользователя по имени через `Telegram.WebApp.initDataUnsafe`.

* **Стриминг данных:** Отправка готового JSON-пакета заказа обратно в чат с ботом с помощью метода `Telegram.WebApp.sendData`.

* **Сценарий 3D-Secure:** Полноценный встроенный симулятор платежного шлюза (ввод карты, лоадер обработки и экран успешной оплаты).

### 🛠 Стек технологий:

* **Frontend:** Vanilla JavaScript (чистый ES6+ модульный код)

* **Стилизация:** Tailwind CSS (via CDN)

* **Интеграция:** Telegram Web Apps API v7+

* **Хостинг:** Netlify (Безопасное HTTPS соединение)

## English

**Venous King** is a high-conversion, luxury Telegram Mini App designed for premium restaurants, street food brands, and gourmet delivery services. It opens instantly inside Telegram in just one click.

### 🔥 Key Features:

* **Luxury UI/UX:** Built with Royal Deep Burgundy color palette and gold accents.

* **Smooth Animations:** Premium hover effects, micro-interactions, and fluid shopping cart transitions.

* **Telegram Native UI:** Fetches Telegram user profile data automatically.

* **Payment Gateway Simulation:** Realistic check-out screen simulating a 3D-Secure credit card payment with loaders and success states.
